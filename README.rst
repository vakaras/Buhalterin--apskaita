===========
MIF Starter
===========

VU MIF dokumentų XeLaTeX šablonas.

Struktūra::

    ├── common          – tai kas bendra visiems;
    ├── config          – konkretaus dokumento individualizavimas;
    ├── dist            – kompiliavimo katalogas;
    ├── part1           – pirmos dalies tekstas;
    └── partn           – n-osios dalies tekstas.
